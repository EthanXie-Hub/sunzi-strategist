# 贡献指南

感谢你帮助改进孙子·战略诊断器。

## 优先欢迎

- 能暴露当前框架失效点的真实战略问题。
- 对手实际采取过、但当前框架漏掉的反制。
- 七计评分尺度、市场进入关卡与止损阈值修正。
- 对《孙子兵法》原文误用、断章取义或过度迁移的修正。
- 其他 Skill 平台的最小兼容适配。

## 不接受

- 把 Skill 改成鼓励硬刚、报复、欺骗、威胁或违法的工具。
- 只增加古文引用，但不改善判断、行动或止损质量。
- 在没有证据时宣称必胜、精确胜率或保证结果。
- 把所有工作、生活问题都强行套成战争。

## 修改要求

每个重要修改应回答：

1. 它解决了哪个真实失败模式？
2. 哪个事实支持这项改动？
3. 它如何改变用户下一步行动？
4. 哪个反例会推翻这项改动？
5. 如何确认它没有削弱安全、信任和合法边界？

提交前运行：

```bash
python3 ~/.codex/skills/.system/skill-creator/scripts/quick_validate.py \
  ./sunzi-strategist
```
