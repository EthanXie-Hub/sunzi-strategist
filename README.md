# 孙子·战略诊断器（sunzi-strategist）

> 基于《孙子兵法》的现实决策诊断器：输入你的现实难题，它用全书的核心框架帮你判断**局势、胜算、风险与下一步行动**。

不是占卜，不是玄学，不是鸡汤，不是教人算计，不鼓励攻击。它做的事只有一件：**在复杂局势中帮你减少误判、降低消耗、提高胜率。**

## 适合谁、解决什么

只要你的处境满足任一条，它就有用：
- **对抗局**：有限资源 + 信息不全 + 有对手/阻力（商业竞争、被竞品压制、职场博弈、谈判、抢资源）；
- **重大抉择**：高风险、不可逆、信息不全，即使没有明确对手（选 offer、要不要 all in、接不接项目、何时出手）。

典型问题：要不要进这个市场？竞品降价跟不跟？该硬刚还是迂回？我胜算多大？资源不够战线太开怎么收？这场败仗是运气还是我的问题？

**不适合**：写代码、排版、查资料等纯执行任务；没有决定要做的闲聊或纯情绪倾诉。

## 怎么用

直接把难题用大白话丢进来，不需要任何格式：

> "竞品昨天突然降价 30%，老板问我们跟不跟。"
> "手上两个 offer，大厂稳定 vs 创业公司期权，要不要赌一把？"
> "我是新咖啡品牌，怎么跟瑞幸星巴克抢上海市场？"

**默认输出精简版**：判断（含信心等级）→ 决定性未知（+低成本侦察法）→ 怎么打（带透镜标签 + 对手反应推演）→ 止损线 → 翻盘条件 → 接下来三步。

想要更深，说"完整展开 / 五层 / 详细"；要量化对比说"按七计打分"；选方案说"帮我对比 A/B/C"；行动结束后回来说"复盘"。

## 伦理边界

- 「不战而屈人之兵」「必以全争于天下」是底色：**赢得局面，不加害个人**。
- 凡需要欺骗信任你的人、烧掉长期关系/声誉才能换的"胜"，skill 会判为下等的「破」局并给保全打法。
- 涉及真实违规/欺诈/骚扰，它会导向证据链与正式渠道（HR/法务/合规），不出阴招。

## 贡献与反馈

1. 用完一次重大诊断后，按 `feedback/feedback-schema.md` 记录结果（成功/部分/失败）；
2. 行动结束回来跑「复盘」（模式 E + AAR 七问）；
3. 改进建议走 `feedback/learning-loop.md` 流程：人工 review → 版本号合并 → 回归测试，**不接受未经评估的 Prompt 直改**。
