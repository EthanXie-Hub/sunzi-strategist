# 生态位与优化轮次（2026-06-11）

回答"这个 skill 凭什么存在"的四个根本问题，并以此确定优化方向。每轮只改一个主方向，过验证门才合并（SkillOpt 式：把 skill 文档当可优化状态）。

## 四问

### 1. 解决的问题成立吗？
**成立，但要说准它解决的是什么。** 通用大模型在重大决策上有三个系统性弱点：顺着用户说（讨好）、结论不可证伪（没有止损线和翻盘条件）、把信息不足藏在流畅行文里。本仓库的同题 A/B 实测（"搞掉同事"题，同一底座模型同一天）显示：裸模型给出共情+劝导+开放式提问；装载后给出 判断+信心等级 → 决定性未知 → 带透镜的打法 → 止损线 → 翻盘条件。**它解决的不是"缺建议"，是"建议不可检验"。** 孙子兵法是钩子与纪律来源，不是卖点本身。

### 2. 用户为什么要安装？
能自己写 prompt 的人也留不住这套纪律——跨会话、跨平台的**一致性**才是安装理由：装一次，所有高风险决策自动进入"先算后说"模式；伦理护栏让它敢接职场冲突类问题而不出阴招；负向边界保证简单问题不被框架骚扰。属于"一致性型"skill（区别于 docx 那类"能力型"），这类 skill 靠**输出差异可见**传播。

### 3. 有没有独特方法论 / 资产 / 数据 / 工作流 / 展示效果？
- **方法论 ✅**（核心资产）：五层诊断 + 输出契约（判断/信心、决定性未知、止损线、翻盘条件、二阶推演）+ 全胜观伦理护栏。
- **数据 ✅**：framework-ledger 全书台账（约 60 条目，原文锚点 + 核心/边角分级）、七计 10 分制评分尺度、市场五关。
- **工作流 ✅**：输出模式 A–F + AAR 复盘七问 + 可控学习循环。
- **脚本 ⚠️**：仅开发侧（static-checks 40 项 + CI + tag 自动发布）；无用户侧脚本——判断型 skill 的合理状态，但意味着传播必须靠输出展示。
- **展示效果 ❌→本轮修复**：此前 README 无对比展示，"为什么装"不可见。

### 4. 一句话传播钩子？
**「不打气，先算账：每个判断带信心等级，每条建议带止损线。」**
（英文位：Strategy diagnoses with confidence levels, stop-losses, and reversal conditions — not cheerleading.）
钩子打的是通用痛点（AI 太会安慰人），孙子兵法作为识别度与信任背书跟在第二句。

## 横向：同类为什么被收藏

抽样（2026-06 检索）：[strategy-skills-for-claude](https://github.com/aapersh/strategy-skills-for-claude)（21 个 McKinsey 式咨询 skill，卖框架广度）、[advisor-strategy-skill](https://github.com/aivsomkar/advisor-strategy-skill)（升级到更强模型当顾问，卖机制）、[alirezarezvani/claude-skills](https://github.com/alirezarezvani/claude-skills)（337 技能合集、5200+星，卖一站式）、中文区 [claude-code-skills-zh](https://github.com/laolaoshiren/claude-code-skills-zh)（100+ 精选、"复制即装"）、[awesome-agent-skills（libukai）](https://github.com/libukai/awesome-agent-skills)（中文终极指南）。

被收藏的共性：**名字即用途**（decision-toolkit 一眼懂）、**输出有可见 artifact**（看截图就知道差异）、**跨平台兼容明示**、**安装零摩擦**。合集靠广度刷星，单体 skill 靠"输出差异截图 + 钩子"传播。检索未发现成型的"孙子兵法决策诊断"同类——**这个位子目前是空的**。

## 纵向：从哪来，往哪去
全文精读 → 五类台账 → v0.1 五层诊断 → v0.2 强化（止损线/评分表/对手反制）→ v0.3 三层架构+工程化 → v0.4 发布工程。当前解决：可证伪的战略诊断。下一步必须补的是**消费侧**：让没读过仓库的人 10 秒内明白为什么装（本轮），然后是失败模式编码与真实反馈闭环（后两轮）。

## 交叉判断：要抢的生态位
**中文 AI 决策技能里的"军师"位：唯一敢给止损线的战略诊断器。**
不与 McKinsey 框架包拼广度（它们卖框架数量），不与合集拼收录（它们卖一站式），拼**判断纪律的深度 + 孙子兵法的零教育成本品牌 + 工程化可信度**（CI/评估集/发布流水线在非代码类 skill 中罕见）。
