# 全书台账：框架 / 原则 / 技法 / 反模式 / 声音

知识底座与单一事实源。用途：① 需要引用原文锚点时查这里；② 判断某透镜是核心（🟢 进主流程）还是边角（⚪ 按需/可省）；③ 维护者增删透镜时先改这里再改 SKILL.md。

全书一句话：**不是教怎么打仗，而是教怎么不靠打仗也能赢**——把胜负决定点从战场前移到计算、定位和信息上（慎战 → 先胜 → 全胜，暗线是夺取主动权「致人而不致于人」）。

## 一、框架（命名的结构模型）

| 名称 | 一句话 | 何时用 | 原文锚点 | 级别 |
|---|---|---|---|---|
| 五事 | 道·天·地·将·法五个基本面 | 进入博弈前全面扫描 | `一曰道，二曰天，三曰地，四曰将，五曰法` | 🟢 |
| 七计 | 五事变成七项可打分对比 | 自己 vs 对手逐项评分 | `主孰有道？将孰有能？天地孰得？法令孰行？兵众孰强？士卒孰练？赏罚孰明？` | 🟢 |
| 庙算 | 开打前沙盘推演，算多者先赢 | 决策前量化胜算 | `多算胜，少算不胜，而况于无算乎` | 🟢 |
| 诡道十二法 | 以信息不对称塑造对手认知 | 制造预期差、抢主动 | `能而示之不能…攻其无备，出其不意` | 🟢 |
| 全/破 | 完整拿下＞打烂了拿下 | 定义什么算赢得漂亮 | `全国为上，破国次之…不战而屈人之兵，善之善者也` | 🟢 |
| 上兵伐谋 | 谋＞交＞兵＞攻城的手段阶梯 | 选打法从上往下找 | `上兵伐谋，其次伐交，其次伐兵，其下攻城` | 🟢 |
| 知胜有五 | 预判胜负五判据 | 行动前自测该不该打 | `知可以战与不可以战者胜…将能而君不御者胜` | 🟢 |
| 形（军形） | 先立于不败再等破绽 | 守底盘、建护城河 | `先为不可胜，以待敌之可胜` | 🟢 |
| 势/奇正 | 结构性动能；正合奇胜 | 设计"一推就赢"的局 | `以正合，以奇胜…如转圆石于千仞之山者，势也` | 🟢 |
| 虚实 | 我之实击敌之虚 | 选切入点避对方最强处 | `避实而击虚…兵无常势，水无常形` | 🟢 |
| 迂直之计 | 绕远反而最快，患转利 | 正面受阻找非直线路径 | `以迂为直，以患为利` | 🟢 |
| 四治 | 治气/心/力/变控节奏 | 长期对抗管理状态曲线 | `避其锐气，击其惰归，此治气者也` | 🟢 |
| 九变 | 该做的恰恰别做的例外原则 | 反教条提醒 | `途有所不由，军有所不击…君命有所不受` | 🟢 |
| 五间 | 因/内/反/死/生五类情报源 | 系统获取对手真实信息 | `五间俱起，莫知其道…知之必在于反间` | 🟢 |
| 风林火山 | 快/稳/猛/静四种节奏 | 选执行节奏风格 | `其疾如风，其徐如林，侵掠如火，不动如山` | ⚪风味 |
| 度量数称胜 | 地→量化实力比链条 | 量化体量差 | `地生度，度生量，量生数，数生称，称生胜` | ⚪ |
| 六地形 | 通挂支隘险远各配打法 | 军事色彩重的场景分类 | `地形有通者，有挂者，有支者…` | ⚪ |
| 九地 | 九类处境心法 | 留散/重/死地概念即可 | `疾战则存，不疾战则亡者，为死地…死地则战` | ⚪概念 |
| 相敌 | 从征兆反推对手状态 | 读对手行为信号 | `辞卑而益备者，进也；辞强而进驱者，退也` | ⚪模块 |
| 五火 | 五类纵火目标 | 纯军事，几无迁移 | `一曰火人，二曰火积…` | ⚪可省 |

## 二、原则（条件式准绳）

| 名称 | 一句话 | 原文锚点 | 级别 |
|---|---|---|---|
| 慎战 | 代价大且不可逆，不值绝不开打 | `兵者，国之大事…不可不察也`／`非利不动，非得不用，非危不战` | 🟢 |
| 先胜而后求战 | 赢的局先布好再动手 | `胜兵先胜而后求战，败兵先战而后求胜` | 🟢 |
| 不可胜在己 | 不败靠自己，取胜等对手给机会 | `不可胜在己，可胜在敌` | 🟢 |
| 知彼知己 | 信息对称度决定胜率档位 | `知彼知己，百战不殆…不知彼不知己，每战必殆` | 🟢 |
| 拙速 | 宁可粗糙地快，不要精巧地拖 | `兵闻拙速，未睹巧之久也`／`兵贵胜，不贵久` | 🟢 |
| 致人而不致于人 | 让对手按你的节奏动 | `致人而不致于人` | 🟢 |
| 杂于利害 | 永远同时算利和害 | `智者之虑，必杂于利害` | 🟢 |
| 恃吾有以待 | 不赌对手不来，只赌自己有准备 | `无恃其不来，恃吾有以待也` | 🟢 |
| 因敌制胜 | 随对手与形势变化，无固定招 | `能因敌变化而取胜者，谓之神` | 🟢 |
| 合于利而动 | 唯利害定进退，不被情绪/沉没成本绑架 | `合于利而动，不合于利而止` | 🟢 |
| 文武并用 | 恩威并施带人 | `令之以文，齐之以武，是谓必取` | ⚪ |

## 五、作者声音（输出腔调）

| 特征 | 用法 | 样例锚点 |
|---|---|---|
| 对仗格言体 | 点睛时用反义对照句，凑不出不硬写 | `胜兵先胜而后求战，败兵先战而后求胜` |
| 辩证相生 | 分析时主动翻出反面 | `乱生于治，怯生于勇，弱生于强` |
| 冷峻会计 | 用成本/阈值/赢面说话，不煽情 | `十则围之，五则攻之`／`食敌一钟，当吾二十钟` |
| 反英雄主义 | 求稳赢不求妙计 | `善战者，无智名，无勇功` |
| 口令式纪律 | 有序可落地的条目 | `凡…故…是故…不可不察也` |
| 进攻话术下的保守 | 先求不败再求胜 | `先为不可胜，以待敌之可胜`／`非危不战` |
| 自然意象 | 需要让原理可感时 | `兵形象水`／`如转圆石于千仞之山` | 

## 现代场景映射速查

| 孙子 | 商业/职场对应 |
|---|---|
| 庙算/五事七计 | 决策前尽调 + 对手/自身打分卡（量表见 scorecards-and-gates.md） |
| 不战而屈人之兵 | 不打消耗战，用定位/生态/谈判取胜 |
| 伐谋>伐交>伐兵>攻城 | 改对手算计 > 动其联盟 > 正面较量 > 强攻其最强业务线 |
| 形/势/虚实 | 护城河 → 势能结构 → 集中打薄弱细分 |
| 用间/先知 | 低成本侦察：问一线、看招聘、找流失客户聊、小测试 |
| 五危/六败/君三患 | 决策前反模式自检：情绪化、美德过度、组织失能、越权干预 |

注：scorecards-and-gates.md 的评分口径偏商业场景（门店/SKU/渠道）；用于职场/个人决策时按"对应物"换算（如「兵众孰强」→你的时间/技能/人脉储备），不要生搬。
