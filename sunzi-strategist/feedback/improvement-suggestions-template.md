# 改进建议（模板）

> 复制本模板为 `improvement-suggestions-YYYYMMDD.md` 填写。每条建议必须可追溯到案例。

## 建议 N：<一句话>

- **触发案例**：case_id 列表（≥3 个，或 1 个 + 逻辑论证）
- **现象**：skill 现在的行为哪里不对/不够
- **根因猜测**：SKILL.md / ledger / scorecards 的哪一节导致
- **建议改动**：具体到改哪段文字、怎么改（给 diff 级描述）
- **风险**：这个改动可能让什么变差（对照伦理边界与右尺寸原则）
- **验证方式**：合并后用 evals 哪个用例验收
